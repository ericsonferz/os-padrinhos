# Address

## Objetivo

Representa um endereço físico.

Pode ser utilizado por:

- WeddingStage
- Vendor
- CoupleProfile
- Participant

---

# Atributos

- country
- province
- district
- city
- neighborhood
- street
- postalCode
- reference

---

# Responsabilidades

Representar um endereço completo.

---

# Regras

É imutável.

Pode possuir campos opcionais.

Não possui identidade própria.
