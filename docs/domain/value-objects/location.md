# Location

## Objetivo

Representa um local onde uma operação acontece.

---

# Pode representar

- Igreja

- Conservatória

- Salão

- Praia

- Quinta

- Casa

---

# Atributos

- name

- address

- latitude

- longitude

- notes

---

# Responsabilidades

Representar um local físico.

---

# Regras

Pode existir mesmo sem coordenadas.

Address é um Value Object interno.
